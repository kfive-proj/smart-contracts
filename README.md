# Smart Contract

## Requirements
- [Truffle](https://www.trufflesuite.com/docs/truffle/overview)
- [Ganache CLI](https://github.com/trufflesuite/ganache-cli)
- Node: v8.9.4 or later
- [Jest](https://jestjs.io/) or [Mocha](https://mochajs.org/)

## Configuration
- truffle.config.js

## Initialize project
```
truffle init
```

## Compile constract
```
npx truffle compile
```
